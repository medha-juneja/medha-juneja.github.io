Convert FString of UE4 to boost::file::path

I had to take a FString filename from Unreal Engine 4 and convert it to boost::file::path type because the rest of my project was using boost::file::path type. You can take the FString from the blueprint and pass it as a parameter to the function in C++. The following code snippet converts FString of UE4 into boost::file::path. 

void UBoostblueprintLibrary::convertFStringtoBoost(FString unrealstring)
{   
    std::string booststring = std::string(TCHAR_TO_UTF8(*unrealstring)); //converts to std::string
    boost::filesystem::path mystring(booststring);  //assigns booststring to boost::file::path type mystring
}
